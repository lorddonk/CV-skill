# English CV Writing Rules

Use these rules when rewriting raw Chinese or English student information into polished CV language.

## General Style

- Write in professional English.
- Translate Chinese source material into natural English CV language, not literal word-for-word English.
- Use concise, achievement-oriented bullet points.
- Start bullets with strong action verbs.
- Use active voice.
- Prioritize relevance to the target application direction.
- Use quantified results when available.
- Highlight methods, tools, data, models, leadership, teamwork, and outcomes.
- Avoid exaggerated, generic, overly promotional, or emotionally loaded language.
- Remove duplicated, irrelevant, overly casual, or weak content.

## Bullet Formula

Preferred logic:

`Action + Task + Method/Tool/Data/Framework + Result/Impact`

Academic examples:

- Developed a CNN-based image classification model using Python and TensorFlow, achieving 92% accuracy on the test dataset.
- Conducted quantitative analysis using SPSS to examine the relationship between consumer behavior and purchase intention.
- Built a MATLAB simulation model to evaluate system performance under different parameter settings.
- Led a team of 4 students in a Kaggle competition and ranked in the Top 10% among participating teams.

Professional examples:

- Optimized backend processes using Python and SQL, reducing data processing time by 40%.
- Conducted market research on 15 competing products and summarized findings into a strategic report for the product team.
- Analyzed customer data using Excel and SPSS to identify key purchasing patterns and support marketing decisions.
- Coordinated cross-functional communication between design and operations teams, improving project delivery efficiency.

## Action Verbs

Use verbs such as:

- Analytical: Analyzed, Evaluated, Interpreted, Assessed, Modeled, Quantified.
- Technical: Developed, Implemented, Built, Designed, Optimized, Automated, Tested.
- Research: Conducted, Investigated, Reviewed, Synthesized, Examined, Validated.
- Communication: Presented, Coordinated, Collaborated, Communicated, Summarized.
- Leadership: Led, Organized, Managed, Directed, Mentored, Facilitated.
- Support: Supported, Assisted, Prepared, Maintained, Documented.

Avoid weak openings such as:

- Responsible for
- Helped with
- Participated in
- Learned about
- Was involved in

If the student truly only assisted, use `Supported`, `Assisted`, or `Contributed to` and still specify the concrete task.

## Quantification Rules

Use numbers only when provided or safely inferable from source material:

- Accuracy, performance improvement, processing time reduction.
- Number of participants, survey responses, interviews, users, products, competitors, clients, reports, team members.
- Ranking, percentile, award level, scholarship selectivity.
- Budget, revenue, transaction volume, dataset size.

If a bullet would benefit from a number but none is provided, add a note in the quality check, such as: `Consider adding dataset size, model accuracy, or competition ranking if available.`

## Language And Tests

For language scores, write clearly:

- IELTS 7.5
- TOEFL iBT 105
- GRE 325
- GMAT 710
- CET-6, if relevant to the applicant context

Do not inflate test scores or convert scores without the user's instruction.

## Tone Control

Use confident but evidence-based language. Avoid claims like `excellent`, `outstanding`, `highly capable`, or `strong passion` unless supported by concrete evidence. Admissions readers trust specific proof more than broad praise.
